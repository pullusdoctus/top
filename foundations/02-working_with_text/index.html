HTML Elements: Basic Text

This little document aims to show the most basic HTML elements in action. These elements will let you create a basic document quickly, so understanding their use is the first step in any web developer's journey!.

Paragraphs

Paragraphs are the basic way to output text on the browser. I'd even say they're the standard way to do this. Chances are, any text you've ever seen on a webpage is a paragraph!

The way to create a paragraph is by using the `<p>` tag around whatever text you want as the paragraph.

Headings

If you want to create sections within your document, use headings! The slightly bigger text that you see on this document is all headings.

Headings help create visual headers, which is useful for human readers, but it also creates sections that can be detected by software, which makes it useful for formatting and parsing.

The way to create a heading is by using the `<hn>` tag around whatever text you want as the heading. Do note that HTML has six levels of headings, so you must specify the heading level with a number instead of `n` (<h1> creates a Header 1).

Emphasis

If you want to emphasize a certain phrase, you can make the text bold or italic! This emphasis, much like headings, is not just visual, but it also adds semantic emphasis to the text, which can be detected by certain software tools.

The way to create bold text is by wrapping the text in a `<strong>` tag. For italic text, you use `<em>` instead.
